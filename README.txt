The package contains following files:
1. demo_topology : It is the topology file which can be opened from flukes application to create a topology on GENI.
2. pox_controller.py : It is the POX controller script which needs to be copied on /root/pox/pox directory of the controller machine.
                       To start the controller:
                       Enter into /root/pox directoy and run command "./pox.py pox_controller"
